COTIZADOR DE COMPUTADORAS

Selecciona tu motherboard

Aquí se conectan todos los componentes de una computadora.

H410M H

Ideal para uso doméstico, gaming o de oficina.

H510M H

Posee compatibilidad para procesadores de 11va generación y posee conector M.2

B460 - PLUS

No soporta la 11va generación, posee: soporte para Intel Optane, conector D-sub y DVI-D

B560 - PLUS

Soporta la 11va generación, posee 2 conectores M.2 y tiene tecnología Aura Sync.

Z490 - GAMING X AX

Es una placa de alta gama pensada para gaming, posee: compatibilidad para la 11va generación, conectores M.2, soporte para intel optane, Rbg y receptor wifi integrado.

Z590 UD

Soporta la 11va generación, posee: capacidad para conectar 4 DIMMS de ram, displayPort, 3 conectores M.2 y soporte para USB 3.2 GEN1 Tipo - C

Selecciona tu procesador

El precesador es el cerebro de una computadora.

Intel Celeron G5920

Ideal para uso doméstico y de oficina.

Intel Core i3 10100

Es opción versátil, sirve como gama de entrada para todos los ámbitos.

Intel Core i5 10400

Potencia considerable, usados para actividades de gran demanda.

Intel Core i7 10700k

Procesador de gama alta, sirve para trabajos que requieran una potencia grande de procesamiento.

Intel Core i9 10900k

Procesador tope de gama, tiene una potencia extrema, es usado para el procesamiento de datos enorme.

Corsair Vengeance LPX DDR4 16Gb

Corsair Vengeance LPX DDR4 32Gb

Patriot Viper Steel DDR4 8Gb

Patriot Viper Steel DDR4 16Gb

Patriot Viper Steel DDR4 32Gb

Selecciona tu fuente de poder

Es la que brinda la alimentación al equipo (nunca elegir genéricas)

Evga 600w 80plus white

Es una de las mejores fuentes de poder económicas.

Evga 850 BQ 80plus bronce

Es una fuente de poder de 800w semimodular

ThermalTake 600w 80plus white

Fuente de poder buena, bonita y barata.

Selecciona tu disco duro

Es el dispositivo que guarda la información del computador.

Seagate 500gb

Bueno y barato.

Seagate 1Tb

Es lo considerado normal en un equipo.

Western Digital 1Tb

Disco duro de una muy buena marca.

Kingston ssd 240Gb

Es una unidad de estado sólido, posee más velocidad que un disco duro (los programas abrirán más rápdio)

Selecciona tu tarjeta de video

Generalmente usa para ejecutar juegos y trabajos de edición de video e imagenes.

Gt 1030

Gama baja, puede servir para conectar más monitores, posee un bajo rendimiento

Gtx 1050 ti

Gama de entrada, permite jugar juegos no tan demandantes a una calidad moderada, puede servir para edición de videos no tan demandantes

Gtx 1650 oc

Buena opción para todos los ámbitos.

Gtx 1660 oc

Posee un rendimiento medio, es ideal para juegos demandantes y trabajos de edición de todo tipo.

Rtx 2080 oc

Gama alto, ideal para juegos 4k y trabajos de edición demandantes.
